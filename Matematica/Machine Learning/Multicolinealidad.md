# Multicolinealidad

Sea $X=[X^{(1)},...,X^{(n)}]$ un vector de predictores de un [[Modelo|modelo]]. Entonces, si la [[Matriz de covarianza|correlaci√≥n]] $\mathrm{Corr}(X^{(i)},X^{(j)})\approx1$ para algunos $i$ y $j$ arbitrarios, entonces existe multicolinealidad. 

Pueden observarse los [[Eigenvalores|eigenvalores]] de la matriz, y si existe mucha diferencia en magnitud entre ellos, entonces hay un problema de colinealidad.
